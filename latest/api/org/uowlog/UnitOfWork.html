  31.  def incrementPending(): Int

    Record that there is a pending execution for this UnitOfWork.

    Record that there is a pending execution for this UnitOfWork. This should only be used if you're implementing support for carrying the UOW across async boundaries. The UOW will not close automatically until after decrementPending is called with the token returned by this routine. Care should be taken to make sure that decrementPending is eventually called, even if it is called from a finalizer; otherwise the UOW will never close automatically.
